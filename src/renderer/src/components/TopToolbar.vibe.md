# TopToolbar 组件文档

## 用户需求
### 原始需求
> 顶部工具栏组件，用于管理翻译项目的选择、分支切换和上游更新

### 期望效果
- 用户可以方便地切换不同的翻译项目
- 可以管理上游分支和工作分支
- 可以刷新上游分支获取最新更新
- 界面简洁直观，操作便捷

## 实现功能
### 主要功能
- [x] 项目选择：下拉选择不同的翻译项目
- [x] 添加项目：通过文件选择器添加新的翻译项目
- [x] 上游分支管理：选择和刷新上游分支
- [x] 工作分支切换：切换当前工作分支
- [x] 自动初始化：自动加载项目列表和当前状态
- [x] 状态同步：项目切换时自动同步分支信息
